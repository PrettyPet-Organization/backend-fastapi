# backend-fastapi

## Setup and run

### Develop

1. **Clone the repository**
   ```bash
   git clone https://github.com/PrettyPet-Organization/backend-fastapi.git
   cd backend-fastapi
   git checkout develop
   ```
2. **Rename the [`.env.template`](.env.template) file to `.env`**, configure it

3. **Run**

   **If you use `uv`:**
    ```bash
    make docker-run
    ```
